# Exercise 3: Prime Number Checker

## The goal of this exercise is to check if a user-input value is a prime number. 

### Toics Covered: 
- Functions 
- Type Casting 
- For loops
- Boolean logic 
- Conditional logic
- Terminal I/O

### How the program works:
- The program first prompts the user to enter a non-negative whole integer. (positive & greater than 1)
- The program then returns a boolean response back to the terminal.
