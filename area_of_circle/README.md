# Exercise 2: Find Area of Circle  

## The goal of this exercise is to find the area of a circle.

### Topics Covered:
- Functions 
- Math :
    - Import math to use Pi 
    - Rounding
    - Exponentiation
- Float Type Casting
- F-string Implementation

### How it works: 
- The program prompts the user for a radius value input 
- Upon entering, the program computes and outputs the area value rounded to 4 decimal points into the terminal.
